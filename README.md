# X予約投稿ツール 運用手順（チームメンバー向け）

## このツールでできること
記事URLやイベント詳細を貼り付けるだけで、AIがXのポスト文を3案作成。1案選んで予約日時を指定すると、サムネイル画像の生成・保存とスプレッドシートへの予約登録まで自動で行われます。

## 使い方（4ステップ）

1. **初回のみ**: アプリを開くとパスコード入力画面が出ます。管理者から共有されたパスコードと自分の名前を入力してください（次回からは不要です）。
2. **STEP 1**: 記事のURL、またはイベント詳細のテキストを貼り付けて「下書きを生成する」を押します（20〜40秒待ちます）。
3. **STEP 2**: 要約とポスト3案（テーマ訴求型／問いかけ型／ターゲット呼びかけ型）が表示されます。良い案をタップして選択します。
4. **STEP 3-4**: 本文を微修正（文字数カウンターが140を超えると赤くなります）→ 予約日時を選択 →「予約する」。完了画面にシートの行番号とサムネイルが表示されます。

## 知っておくこと

- **文字数**: X無料アカウントの制限に合わせ日本語140文字以内。URLは実際の長さに関係なく23文字として換算されます。
- **予約時間は1時間単位**: 11:30を選んでも「11時台」として登録されます（既存の自動投稿の仕組みに合わせているため）。
- **登録先**: スプレッドシート「Twitter予約投稿ツール_v5」の「tweetリスト」シートに1行追加されます。サムネイルPNGはGoogleドライブ「投稿用画像」フォルダに保存されます。
- **登録後の修正**: 内容を直したい場合はスプレッドシートの該当行を直接編集するか、行を削除して登録し直してください。

## うまくいかないとき

- **「パスコードが正しくない」と出る** → 管理者に最新のパスコードを確認してください。
- **「通信に失敗しました」と出る** → 数十秒待って「再試行」を押してください。記事側のサイトが重い場合があります。
- **記事の内容が反映されていない要約が出る** → 会員限定ページ等は読み取れないことがあります。その場合はイベント詳細のテキストを直接貼り付けてください。
