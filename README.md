# X予約投稿ツール 運用手順（チームメンバー向け）

## このツールでできること
記事URLやイベント詳細、あるいは自分で撮った写真＋一言コメントから、AIがXのポスト文を3案作成。最大3案を選んで予約日時を指定すると、画像の保存とスプレッドシートへの予約登録まで自動で行われます。

## 使い方（4ステップ）

1. **初回のみ**: アプリを開くとパスコード入力画面が出ます。管理者から共有されたパスコードと自分の名前を入力してください（次回からは不要です）。
2. **STEP 1**: 記事のURL、イベント詳細のテキスト、または「写真＋一言コメント」を入れて「下書きを生成する」を押します（20〜40秒待ちます）。
   - 写真・動画の添付は任意。画像は最大4枚（自動圧縮）、または動画1本（10MBまで）。
   - 写真を付けると、AIが写真の内容（被写体・雰囲気）も見てコメントを膨らませた投稿文を作ります。
3. **STEP 2**: 要約とポスト3案（フック型のラベル付き）が表示されます。良い案を**最大3つまで**タップして選択できます。
4. **STEP 3-4**: 選んだ案ごとに本文を微修正（文字数カウンターが140を超えると赤くなります）・予約日時を設定 →「予約する」1回で全案がまとめて登録されます。完了画面にシートの行番号が表示されます。

## 画像の扱い（自動判定）

- **写真・動画を添付した場合** → それが投稿に使われます（サムネイル生成なし）
- **記事URLにOGP画像がある場合** → Xが自動でカード画像を表示するため生成不要（チェックで生成に切替可）
- **どちらも無い場合** → ネイビー×ゴールドのサムネイル画像を自動生成します

## 知っておくこと

- **文字数**: X無料アカウントの制限に合わせ日本語140文字以内。URLは実際の長さに関係なく23文字として換算されます。
- **予約時間は1時間単位**: 11:30を選んでも「11時台」として登録されます（既存の自動投稿の仕組みに合わせているため）。
- **登録先**: スプレッドシート「Twitter予約投稿ツール_v5」の「tweetリスト」シートに1行追加されます。サムネイルPNGはGoogleドライブ「投稿用画像」フォルダに保存されます。
- **登録後の修正**: 内容を直したい場合はスプレッドシートの該当行を直接編集するか、行を削除して登録し直してください。

## うまくいかないとき

- **「パスコードが正しくない」と出る** → 管理者に最新のパスコードを確認してください。
- **「通信に失敗しました」と出る** → 数十秒待って「再試行」を押してください。記事側のサイトが重い場合があります。
- **記事の内容が反映されていない要約が出る** → 会員限定ページ等は読み取れないことがあります。その場合はイベント詳細のテキストを直接貼り付けてください。
